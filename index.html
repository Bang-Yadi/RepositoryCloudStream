⚠️PEMBERITAHUAN PENTING⚠️
Waduh ternyata beberapa device ada yang ngebug (ngga muncul plugin nya kalau pake yang SALIN).
Jika kamu mengalami seperti itu, silahkan hapus data app CloudStream, lalu pilih kembali Repository disini dengan klik Install (bukan klik yang Salin)

Salin atau Install Repository
Jika kamu pilih salin, maka akan menyalin link dan kamu harus masukkan manual ke app CloudStream.
Jika kamu pilih Install, itu akan mendeteksi app CloudStream dan membuka nya.
Akan muncul pilihan Install/Pasang/Download saat masuk ke app CloudStream nya.
